# Objetivo

Trabalho de refatoração feito na UC de Gestão e Qualidade de Software. O principal objetivo do trabalho foi refatorar um código legado aplicando boas práticas do Clean Code, como 
SOLID, DRY, KISS, etc.

# Como rodar

Baixe o repositório e abra o projeto no NetBeans. Importe o banco de dados dentro da pasta database no MySQL.
